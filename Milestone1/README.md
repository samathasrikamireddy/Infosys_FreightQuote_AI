# Infosys Springboard Internship 7.0

## Milestone 1 – User Authentication Module

### Project Description
This project is developed as part of the Infosys Springboard Internship 7.0. It implements a secure User Authentication Module using Streamlit with JWT authentication, Gmail OTP verification, and ngrok.

### Features
- User Login
- User Signup
- Forgot Password using Security Question
- Forgot Password using OTP
- JWT-based Session Management
- User Dashboard
- Admin Dashboard

### Technologies Used
- Python
- Streamlit
- JWT
- Gmail SMTP
- ngrok
- Google Colab

### How to Run
1. Open the notebook in Google Colab.
2. Install the required libraries.
3. Add the required secrets (JWT, ngrok, Gmail).
4. Run all notebook cells.
5. Open the generated ngrok URL.
